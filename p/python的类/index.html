面向过程和面向对象

面向过程【POP】是一种以过程为中心的编程思想，将程序看作一系列的命令，函数或过程的集合，主要是解决问的步骤和过程。程序被划分为一系列的函数，每个函数扶着完成一个特定的任务。数据在整个程序都是自由流动的，任何函数都可以访问和修改数据。

面向对象编程【OOP】是将程序看作一系列对象的集合，每个对象都是独立 的个体，包含数据和操作这些数据的方法。这些对象之间通过发送消息来相互共同解决某种问题。

面向过程：
优点：效率高，性能高效。
缺点：耦合度高，扩展性差，不易维护。

面向对象：
优点：模块化，扩展性强，易维护，代码重复性高。
缺点：效率比面向过程低，学习难度大，性能开销大。

类的定义

在Python中，使用class关键字定义一个类。定义格式为：

class MyClass:

类的调用，也称为类的实例化，实例化出来的内容就称为对象，也叫创建对象。

创建对象的格式为：

对象名 = 类名（）

对象与类之间的关系：

  1. 对象拥有类所定义的全部的属性和行为。
  2. 对象的属性和行为可以单独进行增加、修改、删除。
  3. 对象不能单独创建，必须依托类的实例化，且一个类可以实例化无数的对象。
  4. 对象之间的属性和行为不是共享的。

self参数

self是一个参数，表示对象自身，里面存放着对象自身的地址。如果希望类中的方法可以被对象调用，那么第一个参数必须是self。作用就是将实例对象与类的方法进行绑定，这样才能让每个对象都能调用属于自己的方法。

类的封装

在Python中可以通过给属性名和方法名添加下划线（_）设置权限，根据下划线的个数分为不同类型：

  1. 单下划线前缀：表示该属性或方法是内部使用，事实上程序是可访问的。
  2. 双下划线前缀：设置私有权限的方法，将该属性或行为定义为私有，且不可被对象访问。
  3. 双下划线前后缀：表示Python中的特殊的属性或方法，有特殊的意义和用途，不推荐自定义。

#私有权限设置
class Person:
    def __init__(self,name,password):
        self.name = name
        self.phone_id= '1234567890'
        self.__password = None
    #设置一个私有属性
    def __say(self):
        print(f"我的密码是{self.__password}")

    def change_password(self):
        self.__password = "4567898"
        self.__say()

p1 = Person('zhangsan','123456')
print(p1.name)
print(p1.phone_id)
p1.change_password()

类的继承

在Python中，一个类可以被继承，也可以去继承别的类，其中类的关系为父类与子类。在继承过程中，子类会继承父类的所有属性和行为，并且一个子类可以有多个父类。

表示格式为：

class Person(要继承的类):
    pass

当不写【要继承的类】时，Python3.x中默认继承object类。object类中有以下函数：

名称 作用
new() 创建对象
init() 初始化对象
eq() 定义比较符
iter() 让对象支持迭代
next() 在迭代中返回下一个值
class 对象所属的类
doc 对象的文档字符串
name 类的名称

类的多继承中，可以继承多个父类，但是也可能导致冲突，比如子类和多个父类有相同的属性和方法。那么此时在使用该属性或方法，其顺序为：

子类》从左到右第一个父类》第二个父类》...

对于复杂的继承关系，使用子类的mro( )方法获取继承顺序。

#多继承
class A:
    def __init__(self):
        print("A")

class B1(A):
    def __init__(self):
        A.__init__(self)
        print("B1")

class B2(A):
    def __init__(self):
        A.__init__(self)
        print("B2")

class C(B1,B2):
    def __init__(self):
        B1.__init__(self)
        B2.__init__(self)
        print("C")

c = C()
#打印结果
A
B1
A
B2
C

这里的打印结果中，出现了两次A，如何避免这种资源浪费？

答案是：使用super() 函数，根据mro继承顺序去搜索父类中的指定函数，并且自动绑定self参数。

#多继承，使用super()函数
class A:
    def __init__(self):
        print("A")

class B1(A):
    def __init__(self):
        super().__init__()
        print("B1")

class B2(A):
    def __init__(self):
        super().__init__()
        print("B2")

class C(B1,B2):
    def __init__(self):
        super().__init__()
        print("C")

c = C()
#打印结果
A
B2
B1
C

类的多态

多态指允许不同对象对同一个方法卓出不同响应，即同一个方法可以根据不同对象的类型而表现出不同行为。

比如len()函数，对于字符串来说是统计字符串的长度，对于列表是统计列表中元素的个数。

在类的继承中所用到的方法重写，就是一种实现多态的方式。通过重写父类中的属性和方法，使其拥有自己独特的属性和方法。
